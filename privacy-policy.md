# Privacy Policy for ID Finder - Chrome & Edge Extension

**Effective Date:** [10 April 2026]

This Privacy Policy describes how your personal information is handled in the ID Finder Extension ("the Extension").

## Information Collection
**The ID Finder Extension does not collect, use, store, or transmit any personal user data.**

The Extension's sole function is to inspect the Document Object Model (DOM) of the webpage you are actively viewing. All inspection happens locally within your browser instance. No information is sent to any external server or third-party service.

## Data Usage
Since the Extension does not collect any data, it does not share, sell, or transfer any user information to third parties.

## Permissions Justification
The Extension requires the following permissions only to provide its core functionality:
- **`activeTab`**: This permission is needed to access and read the content (the HTML elements and their IDs) of the currently active browser tab when you activate the Extension.
- **`clipboardWrite`**: This permission is needed to copy an element's ID to your system clipboard when you click on it.

## Changes to This Privacy Policy
Any future changes to this Privacy Policy will be posted on this page. You are advised to review this Privacy Policy periodically for any changes.
